Visual Studio学习日志

发表于 2017-12-05

颜色主题

打开【工具】>【选项】，在【环境】>【常规】中【颜色主题】可改为深色。

Color Scheme

Studio Styles提供了很多Color Scheme
https://studiostyl.es/
选择喜欢的Color Scheme，选择对应的版本下载。
打开【工具】>【导入和导出设置】，选择【导入选定的环境设置】

视需要备份当前设置；
选择【浏览】打开下载的Color Scheme。

自动排版

ctrl+a 全选
编辑.格式化选定内容 ctrl+k，ctrl+f 根据周围的代码行，正确缩进选定的代码行。

注释与去掉注释

编辑.注释选定内容 ctrl+k，ctrl+c 使用编程语言的正确注释语法将代码的当前行标记为注释。
编辑.取消注释选定内容 ctrl+k，ctrl+u 从代码的当前行中移除注释语法。

显示解决方案资源管理器

视图.解决方案资源管理器 ctrl+alt+l 显示解决方案资源管理器。

快捷键

撤消 ctrl+z
重复 ctrl+y
转换为大写 ctrl+shift+u
转换为小写 ctrl+u
开始执行（不调试）ctrl+f5
快速隐藏或显示当前代码段 ctrl+m+m （按两次m）
